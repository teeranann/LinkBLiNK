# Example data

Each folder holds the first 50 frames of one video, enough to run the LinkBLiNK pipeline end-to-end as a smoke test. The full benchmark datasets (Scenarios A, B, BI, BP, BE, BX and the complete real videos) are available from the corresponding author on reasonable request, as stated in the manuscript Data Availability section.

```
real/Real Particle 1/   real F8BT nanoparticle video
real/Real Particle 2/   real F8BT nanoparticle video
synthetic/ScenarioA/    simulated reconnection scenario
synthetic/ScenarioB/    simulated impostor-rejection scenario
```
